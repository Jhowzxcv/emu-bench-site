# Emu Bench — site

Landing page do **Emu Bench**, a central de emulação premium pra Windows
(grátis e open-source). Site estático servido via GitHub Pages.

- **Download / Releases:** https://github.com/Jhowzxcv/emu-bench-releases/releases
- **Política de privacidade:** [privacy.html](privacy.html)

> O Emu Bench não distribui jogos, BIOS, firmware ou chaves (keys) — apenas
> gerencia, testa e organiza o conteúdo que você já possui legalmente.
